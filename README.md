Step 1: Check Python Version
Open a terminal and confirm Python 3.8+ is installed:
bashpython --version
If not installed, download from python.org.

Step 2: Install Dependencies
Run this single command to install everything needed:
bashpip install opencv-python mediapipe numpy Pillow

On some systems use pip3 instead of pip.


Step 3: Run the Program
bashpython gesture_3d_designer.py
A window titled "🎨 Gesture 3D Designer" will open — a white canvas with a small webcam preview in the bottom-right corner.

Step 4: Position Your Camera

Sit 1–2 feet from your webcam in good lighting (face a window or lamp)
Make sure your hand is clearly visible against a plain background

Tips for Best Results

Change colors first — pinch and drag your hand up/down before painting
Layer effects — paint first, then add 3D text on top
Move slowly for clean trails; fast movement = wild splatter
The color swatch (top-left square) always shows your current color
Press Q or Esc to quit and save nothing (screenshot with Win+Shift+S / Cmd+Shift+4 to save your art!)# -Gesture-3D-Design
